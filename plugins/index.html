This directory lists plugins that either extend Rundeck or plugins or services that allow other tools to integrate with Rundeck. Need a plugin? Please write to us about any plugin you wish was here. Interested in your own plugin appearing here? Take a look at the Plugin Publishing Guide.

Rundeck can integrate with tools in several manners:

  1. Rundeck Plugin. A Rundeck plugin implemented in Java or shell script, installed into a Rundeck server.
  2. External Service. An external service that is used by a Rundeck plugin or the Rundeck core.
  3. External Plugin. A plugin installed in another tool that interacts with Rundeck through its API.

AWS EC2 Nodes

  • Category : aws
  • Rundeck Plugin: Resource Model Source
  • Rundeck version : 1.5 +

This plugin contains a Resource Model Source provider that returns Amazon EC2 Instances as nodes to the RunDeck server.

Chef Nodes

  • Category : chef
  • External Service: URL Resource Model Source
  • Rundeck version : 1.3 +

Use this web service to provide Chef node data to Rundeck projects. Set up the web service and then configure your project to access the Chef data via Rundeck's built-in URL Resource Model Source.

Jenkins-Rundeck

  • Category : jenkins
  • External Plugin: Jenkins
  • API version : 1.3 +

This plugin is a Notifier that will talk to a RunDeck instance (via its HTTP API) to schedule a job execution on RunDeck after a successful build on Jenkins. It is also a Trigger that will schedule a build on Jenkins after a job execution on RunDeck (using RunDeck WebHook Notification). In addition, it turns Jenkins into an Option provider for RunDeck, if you want to use your Jenkins build artifacts as an option to a RunDeck job.

Nexus-Rundeck

  • Category : nexus
  • External Plugin: Nexus
  • API version : 1.3 +

This Nexus plugin turns Nexus into an Option provider for RunDeck, if you want to use your Nexus artifacts as an option to a RunDeck job. The plugin provides the following new HTTP resources :

  • http://NEXUS_HOST/service/local/rundeck/options/version : return a json array with the version of the matching artifacts.
  • http://NEXUS_HOST/service/local/rundeck/options/artifactId : return a json array with the artifactId of the matching artifacts.
