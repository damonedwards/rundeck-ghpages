This directory lists plugins that either extend Rundeck or plugins or services that allow other tools to integrate with Rundeck.

  • Need a plugin? Please write to us about any plugin you wish was here.
  • Have a plugin? Want your plugin to appear here? Take a look at the Plugin Publishing Guide.

Additional plugins not listed here can be found at:

  • http://github.com/rundeck-plugins/

Rundeck can integrate with tools in several manners:

  1. Rundeck Plugin. A Rundeck plugin implemented in Java or shell script, installed into a Rundeck server.
  2. External Service. An external service that is used by a Rundeck plugin or the Rundeck core.
  3. External Plugin. A plugin installed in another tool that interacts with Rundeck through its API.
