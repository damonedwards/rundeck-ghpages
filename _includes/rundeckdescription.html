Rundeck is an open-source workload automation server with a graphical console and web API.

Define multi-step workflows to automate your processes. Workflow steps can dispatch shell commands and scripts across all your physical or virtual nodes and execute actions in other tools through plugins. Use Rundeck as a portal to provide self serve automation or use Rundeck as an automation engine to power your own hosted service. Configure extensive role-based access control with LDAP/Active Directory and policies.
