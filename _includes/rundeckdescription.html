Rundeck is an open-source workflow automation server for DevOps and the Cloud.

Define multi-step workflows to automate your processes across physical and virtual infrastructure. Pair Rundeck with popular configuraiton management tools like Puppet and Chef to deploy your applications. Use Rundeck's web UI as a portal to create self service automation. Build your own automation solution using Rundeck's web API. Configure extensive role-based access control with LDAP/Active Directory integration.
